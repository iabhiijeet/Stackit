# StackIt 🚀

StackIt is a modern collaborative task management platform inspired by Trello.  
It helps teams and individuals organize tasks, manage workflows, and collaborate efficiently using Kanban-style boards.

---

## ✨ Features

- 📋 Create and manage multiple boards
- 🧩 Drag & drop task cards
- 👥 Team collaboration
- 🔐 Authentication & Authorization
- 🕒 Real-time updates
- 📌 Labels and priorities
- 📅 Due dates and reminders
- 💬 Task comments and activity logs
- 📎 File attachments
- 🌙 Dark/Light mode
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Next.js
- Tailwind CSS
- Redux Toolkit / Context API

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT
- bcrypt.js

### Realtime
- Socket.io
